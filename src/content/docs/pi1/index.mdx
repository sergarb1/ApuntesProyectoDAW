## ¿Qué es PI1?

PI1 es la primera parte del Proyecto Intermodular de DAW. Se desarrolla
durante el primer curso y consta de **5 retos** que te guían desde la
identificación de una necesidad hasta la presentación de un MVP.

### Los 5 retos

| Reto | Qué harás | Entregable |
|------|-----------|------------|
| 🔍 **R1: Identificación** | Definir el problema y la idea de solución | Documento de definición |
| 📝 **R2: Análisis** | Analizar requisitos y planificar | Documento de análisis |
| ✏️ **R3: Diseño** | Diseñar la arquitectura y la interfaz | Documento de diseño |
| ⚙️ **R4: Implementación** | Construir el MVP | Código funcional |
| 🎤 **R5: Presentación** | Defender el proyecto | Presentación + memoria |

### Evaluación

Cada reto se evalúa de forma independiente. La nota final es la media
ponderada según el peso de cada reto (consultar guía de evaluación).
