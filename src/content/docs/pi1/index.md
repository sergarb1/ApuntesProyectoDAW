## Objetivo del Módulo

**Proyecto Intermodular I** se cursa en 1º de DAW. El objetivo fundamental es que el alumno sea capaz de **transformar una idea en un producto mínimo viable (MVP) funcional**, aplicando los conocimientos adquiridos en los módulos de primer curso.

### Enfoque

Trabajarás siguiendo una metodología de aprendizaje basado en proyectos, dividido en **5 retos** consecutivos que te guiarán desde la identificación de un problema hasta la presentación de un prototipo funcional.

### Módulos Relacionados

| Módulo | Contribución al Proyecto |
|---|---|
| Programación | Lógica de negocio, estructuras de datos, algoritmos |
| Bases de Datos | Diseño e implementación del modelo de datos |
| Lenguajes de Marcas | XML, JSON, estructuras de datos intercambiables |
| Entornos de Desarrollo | Uso de IDE, depuración, control de versiones |
| Sistemas Informáticos | Instalación y configuración de entornos |
| Digitalización | Herramientas ofimáticas y digitales |

### Resultados de Aprendizaje

Al finalizar PI1, el alumno será capaz de:

- Identificar una necesidad real y proponer una solución software viable
- Analizar requisitos funcionales y documentarlos con historias de usuario
- Diseñar la arquitectura básica de una aplicación web
- Implementar un MVP funcional que resuelva el problema planteado
- Presentar y defender el proyecto ante un tribunal
