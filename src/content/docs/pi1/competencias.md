## Competencias del Módulo

En PI1 se trabajan las competencias asociadas a los módulos de primer curso del ciclo DAW. Cada una de ellas se refleja en el proyecto a través de entregables y criterios de evaluación específicos.

## Programación

- Implementar la lógica de negocio de la aplicación
- Manejar estructuras de datos (arrays, listas, mapas, objetos)
- Aplicar principios de programación estructurada y modular
- Utilizar control de versiones para el código fuente

## Bases de Datos

- Diseñar el modelo entidad-relación del dominio del problema
- Implementar el esquema en un SGBD (MySQL, PostgreSQL, SQLite)
- Crear consultas SQL para la manipulación de datos
- Conectar la aplicación con la base de datos

## Lenguajes de Marcas

- Utilizar XML para el intercambio de datos entre sistemas
- Manejar JSON para la comunicación con APIs
- Validar documentos mediante esquemas (XSD, JSON Schema)

## Entornos de Desarrollo

- Configurar y utilizar un IDE profesional (VS Code, IntelliJ)
- Depurar código utilizando herramientas del entorno
- Gestionar el proyecto con un sistema de control de versiones (Git)
- Documentar el código siguiendo estándares

## Sistemas Informáticos

- Configurar el entorno de desarrollo local
- Instalar y administrar servidores web y de bases de datos
- Comprender la arquitectura cliente-servidor

## Digitalización

- Elaborar documentación técnica con herramientas ofimáticas
- Crear presentaciones profesionales
- Utilizar herramientas colaborativas en la nube
