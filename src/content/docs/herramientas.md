## Herramientas del Curso

A continuación se presentan las herramientas recomendadas para el desarrollo de los proyectos intermodulares. Cada herramienta tiene un propósito específico dentro del flujo de trabajo.

<CardGrid>
  <Card title="Git">
    Sistema de control de versiones distribuido. Permite gestionar el código fuente, mantener el historial de cambios y colaborar con otros desarrolladores.
  </Card>
  <Card title="GitHub">
    Plataforma de alojamiento de repositorios Git. Ofrece issues, pull requests, actions para CI/CD, projects y wikis para documentación colaborativa.
  </Card>
  <Card title="Jira">
    Herramienta de gestión de proyectos ágiles. Ideal para la planificación de sprints, gestión del backlog y seguimiento de tareas con Scrum o Kanban.
  </Card>
  <Card title="Trello">
    Alternativa ligera a Jira para gestión visual de tareas mediante tableros Kanban. Útil para proyectos pequeños o gestión personal del trabajo.
  </Card>
  <Card title="Draw.io">
    Herramienta gratuita de diagramación. Permite crear diagramas UML, modelos E/R, diagramas de flujo y wireframes directamente en el navegador.
  </Card>
  <Card title="Figma">
    Herramienta de diseño de interfaces colaborativa. Ideal para crear prototipos, wireframes interactivos y diseñar la experiencia de usuario.
  </Card>
  <Card title="Docker">
    Plataforma de contenerización. Permite empaquetar la aplicación con todas sus dependencias para ejecutarla de forma consistente en cualquier entorno.
  </Card>
  <Card title="Railway">
    Plataforma cloud para despliegue de aplicaciones. Ofrece despliegue sencillo con integración directa desde GitHub y base de datos PostgreSQL incluida.
  </Card>
  <Card title="Vercel">
    Plataforma de despliegue optimizada para frontend. Despliegue automático desde GitHub, soporte para frameworks modernos y funciones serverless.
  </Card>
  <Card title="VS Code">
    Entorno de desarrollo integrado (IDE). Editor ligero y extensible con soporte para todos los lenguajes del ciclo, integración Git y terminal incorporada.
  </Card>
</CardGrid>

## Stack Tecnológico Recomendado

| Capa | Tecnología | Alternativas |
|---|---|---|
| Frontend | React + Vite | Vue 3, Angular, Svelte |
| Backend | Node.js + Express | Laravel, Django, Spring Boot |
| Base de Datos | PostgreSQL | MySQL, SQLite, MongoDB |
| ORM | Prisma / Sequelize | Doctrine, Hibernate |
| Testing | Vitest + Playwright | Jest, PHPUnit, Cypress |
| Contenedores | Docker + Docker Compose | Podman |
| CI/CD | GitHub Actions | GitLab CI, Jenkins |
| Cloud | Railway + Vercel | Netlify, Render, Fly.io |
| Diseño | Figma | Penpot, Sketch |
| Documentación API | Swagger / OpenAPI | Postman, Redoc |
