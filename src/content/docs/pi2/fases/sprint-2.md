title: Sprint 2 - Frontend Profesional
description: Desarrollo de la interfaz de usuario con framework moderno
---

## Objetivo

Reimplementar la interfaz de usuario utilizando un framework moderno, aplicando principios de diseño profesional, usabilidad y accesibilidad.

## Entregables

### 1. Framework de Frontend

Implementar la interfaz con uno de los siguientes frameworks:

- **React** con Vite o Next.js
- **Vue 3** con Composition API
- **Angular** standalone
- **Svelte** con SvelteKit

### 2. Diseño Responsive

La aplicación debe funcionar correctamente en:

- **Móvil**: 320px - 768px
- **Tablet**: 768px - 1024px
- **Escritorio**: 1024px o más

### 3. Componentes Clave

Desarrollar como mínimo:

- Sistema de navegación (menú responsive)
- Formularios con validación en cliente
- Tablas con filtros y ordenación
- Modales o paneles laterales
- Indicadores de carga y estados vacíos
- Sistema de notificaciones (toast, alertas)

### 4. Gestión de Estado

Implementar gestión de estado para:

- Autenticación del usuario
- Datos de sesión
- Preferencias de usuario (tema, idioma)
- Cache de datos del servidor

### 5. Consumo de API

- Cliente HTTP configurado (Axios, Fetch)
- Manejo de errores y timeouts
- Interceptor de autenticación (token JWT)

## Criterios de Evaluación

- Framework correctamente configurado y estructurado
- Diseño responsive funcional en todos los tamaños
- Componentes reutilizables y bien diseñados
- Validación de formularios en cliente operativa
- Consumo de API correctamente implementado
- Código limpio y organizado (ESLint, Prettier)
