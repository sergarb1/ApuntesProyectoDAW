## Competencias del Módulo

En PI2 se trabajan las competencias asociadas a los módulos de segundo curso del ciclo DAW. Estas competencias se evalúan de forma integrada a través del proyecto.

## DWEC (Desarrollo Web en Entorno Cliente)

- Desarrollar interfaces de usuario interactivas con JavaScript moderno
- Utilizar frameworks de frontend (React, Vue, Angular o Svelte)
- Implementar comunicación asíncrona con el servidor (AJAX, Fetch API)
- Gestionar el estado de la aplicación en el cliente
- Validar formularios y datos en el lado del cliente

## DWES (Desarrollo Web en Entorno Servidor)

- Diseñar e implementar APIs RESTful
- Gestionar sesiones, autenticación y autorización
- Implementar la lógica de negocio completa del sistema
- Manejar operaciones CRUD complejas con base de datos
- Aplicar patrones de diseño en el backend (MVC, DAO, Repository)

## DIW (Diseño de Interfaces Web)

- Aplicar principios de diseño UI/UX en la interfaz
- Implementar diseño responsive adaptado a múltiples dispositivos
- Garantizar la accesibilidad (WCAG) de la aplicación
- Crear una experiencia de usuario coherente y profesional

## DAW (Despliegue de Aplicaciones Web)

- Configurar servidores web (Apache, Nginx)
- Contenerizar la aplicación con Docker
- Desplegar en plataformas cloud (Vercel, Railway, AWS)
- Automatizar despliegues con CI/CD

## Empresa e Iniciativa Emprendedora

- Definir el modelo de negocio del proyecto
- Realizar un análisis de viabilidad técnica y económica
- Aplicar licencias de software y propiedad intelectual

## Proyecto Intermodular

- Integrar todos los conocimientos del ciclo en un producto final
- Documentar el proyecto con estándares profesionales
- Defender el proyecto ante un tribunal técnico
